## 🏗️ Raymake: 现实模拟的终极推演

**Raymake** 是一款致力于深度还原现实逻辑的模拟游戏。我们不满足于简单的数值堆砌，而是试图构建一个从**个体微观生活**到**国家宏观外交**的全维度模拟系统。

> **“在这里，每一个决策都在重塑社会的肌理。”**

---

## 🌟 核心愿景 (Core Vision)

我们的目标是打造一个闭环的模拟世界，涵盖以下核心层级：

- **🧱 微观基石：** 深度模拟。每一个“居民”都有其生存需求与行为逻辑。
- **⚖️ 社会契约：** 构建社区、城市与社会分层。
- **📈 宏观秩序：** 完整的经济系统与外交机制，涉及通货膨胀、贸易博弈及国际局势推演。
- **🌍 演化共生：** 让世界拥有生命力。
